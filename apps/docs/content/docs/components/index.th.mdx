ชุดคอมโพเนนต์ของ Kairo ครอบคลุมห้าหมวดหมู่: ฟอร์ม, นำทาง, แสดงผล, ย่อ/ขยาย
และซ้อนทับ รวมทั้งหมด 28 คอมโพเนนต์ คอมโพเนนต์ที่โต้ตอบได้ทุกตัวเป็น wrapper
บาง ๆ ของ Base UI primitive จัดสไตล์ด้วย CSS ธรรมดาที่มาจาก `@kairo-ui/theme`
ทั้งหมด — ไม่ต้องใช้ Tailwind หรือ CSS-in-JS runtime ในแอปของคุณ
เลือกดูตามหมวดหมู่ด้านล่าง หรือกดไปที่คอมโพเนนต์ที่ต้องการได้เลย

## ฟอร์ม

| Component                                          | คำอธิบาย                                                                                                                          |
| --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| [Button](/th/docs/components/button)                 | อีลีเมนต์สำหรับคลิกที่มีสามรูปแบบ solid, outline และ ghost ในสามขนาด                                                                  |
| [Input](/th/docs/components/input)                   | text input ดั้งเดิมที่จัดสไตล์ด้วยธีมของ Kairo รองรับสถานะปิดใช้งานและไม่ถูกต้อง                                                      |
| [NumberField](/th/docs/components/number-field)       | อินพุตตัวเลขที่รองรับ locale พร้อมปุ่ม stepper สำหรับเพิ่ม/ลดค่า สร้างบน Base UI                                                      |
| [Select](/th/docs/components/select)                 | listbox แบบ dropdown สำหรับเลือกค่าจากรายการที่กำหนดไว้ล่วงหน้า สร้างบน Base UI                                                       |
| [Combobox](/th/docs/components/combobox)             | ช่องข้อความที่จับคู่กับ listbox แบบกรองได้ สำหรับเลือกค่าจากรายการที่ค้นหาได้ สร้างบน Base UI                                          |
| [Checkbox](/th/docs/components/checkbox)             | ตัวควบคุมสามสถานะ (checked/unchecked/indeterminate) สร้างบน Base UI                                                                  |
| [RadioGroup](/th/docs/components/radio-group)         | กลุ่มตัวเลือกที่แยกจากกัน เลือกได้ครั้งละหนึ่งตัวเท่านั้น สร้างบน Base UI                                                              |
| [Switch](/th/docs/components/switch)                 | ตัวควบคุมแบบสลับสำหรับสถานะเปิด/ปิด สร้างบน Base UI                                                                                   |
| [ToggleGroup](/th/docs/components/toggle-group)       | ปุ่มสองสถานะที่ค้างการกดไว้ได้ ใช้เดี่ยว ๆ หรือจัดกลุ่มเป็น ToggleGroup แบบเลือกได้ทีละรายการหรือหลายรายการ สร้างบน Base UI            |
| [Slider](/th/docs/components/slider)                 | ตัวควบคุมแบบลากที่ใช้เลือกค่าตัวเลข หรือช่วงค่าสองค่า ไปตามแทร็ก สร้างบน Base UI                                                       |

## นำทาง

| Component                       | คำอธิบาย                                                                                              |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| [Tabs](/th/docs/components/tabs)   | จัดระเบียบเนื้อหาเป็นพาเนลที่สลับได้ พร้อมตัวชี้แท็บที่ active แบบมีแอนิเมชัน สร้างบน Base UI               |

## แสดงผล

| Component                                | คำอธิบาย                                                                                                                                                    |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Badge](/th/docs/components/badge)           | ป้ายเล็ก ๆ สำหรับแสดงสถานะ จำนวน และ metadata มีสามรูปแบบ solid, soft และ outline ครอบคลุมห้า intent                                                              |
| [Avatar](/th/docs/components/avatar)         | แสดงรูปโปรไฟล์ของผู้ใช้ โดยตกกลับไปแสดงอักษรย่อหรือข้อความโดยอัตโนมัติ สร้างบน Base UI                                                                            |
| [Card](/th/docs/components/card)             | คอนเทนเนอร์สำหรับจัดกลุ่มเนื้อหาที่เกี่ยวข้องกัน ประกอบจากส่วน Header, Title, Description, Content และ Footer                                                     |
| [Spinner](/th/docs/components/spinner)       | ตัวบ่งชี้การโหลดทรงกลมที่ปรับสีตามสีข้อความโดยรอบ                                                                                                                  |
| [Separator](/th/docs/components/separator)   | เส้นบาง ๆ ที่แบ่งเนื้อหาทั้งในเชิงภาพและเชิงความหมาย รองรับทั้งแนวนอนและแนวตั้ง                                                                                    |
| [Progress](/th/docs/components/progress)     | แถบแสดงความคืบหน้าแบบ static ที่มีสถานะทั้งแบบทราบค่าและไม่ทราบค่า สร้างตามรูปแบบ WAI-ARIA progressbar                                                            |
| [Meter](/th/docs/components/meter)           | เกจแบบสถิตสำหรับปริมาณที่รู้ค่าและมีขอบเขตแน่นอน — การใช้งานดิสก์ ความแข็งแรงของรหัสผ่าน โควตาพื้นที่จัดเก็บ — พร้อมเกณฑ์ low/high/optimum ที่ปรับสีของแถบเติมให้อัตโนมัติ |
